# Component Guide

## animations/
- FadeIn - wraps any element with a scroll-triggered fade + slide
- StaggerChildren + StaggerItem - staggered list reveals

## sections/ (reused across pages)
- HeroSection - homepage hero with search
- StatsSection - 4-stat counter bar
- CTASection - customizable call-to-action banner

## shared/ (utility UI)
- SectionHeader - badge + title + description block
- PageHeader - full-width page title banner

## cards/
- JobCard - displays a single job listing
- BlogCard - displays a blog post preview

## forms/
- ContactForm - validated contact form with RHF + Zod

## layout/
- Navbar - sticky header with mobile menu
- Footer - link columns + copyright
- ThemeProvider - wraps app for dark mode
